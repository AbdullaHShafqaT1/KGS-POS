# 🚀 Complete Deployment Guide for Windows

## Step-by-Step Installation & Setup

### **Step 1: Extract the ZIP File**

1. Download `Karachi-Garments-Store-POS-v1.0.0.zip` from the GitHub releases or dist-electron folder
2. Right-click the ZIP file
3. Select **"Extract All..."**
4. Choose a location (recommended: `C:\Program Files\` or `C:\Users\YourUsername\AppData\Local\`)
5. Click **Extract**
6. Wait for extraction to complete

**Result**: You'll have a folder named `Karachi-Garments-Store-POS-v1.0.0` with all files inside

---

### **Step 2: Verify the Contents**

Navigate to the extracted folder and check:

```
Karachi-Garments-Store-POS-v1.0.0/
├── Karachi Garment Store POS.exe    ← Main executable
├── resources/
│   ├── app/
│   └── ... (app files)
├── node_modules/
├── locales/
└── ... (other files)
```

✅ The **`Karachi Garment Store POS.exe`** file must be present

---

### **Step 3: Test the Application**

1. Double-click **`Karachi Garment Store POS.exe`**
2. Wait 3-5 seconds for the app to start
3. You should see:
   - Animated splash screen with "Karachi Garments Store"
   - Login page appears
   - Barcode server starts (check terminal/console)

**Verify**:
- ✅ Login works (use SUPadmin / SUP!@#123)
- ✅ Can add products
- ✅ Can create bills
- ✅ Reports display correctly

---

### **Step 4: Create Desktop Shortcut**

#### **Method A: Using Windows Right-Click Menu (Easiest)**

1. Navigate to the extracted folder
2. Find **`Karachi Garment Store POS.exe`**
3. Right-click on it
4. Select **"Send to" → "Desktop (create shortcut)"**
5. A shortcut will appear on your desktop

✅ **Done!** Double-click the shortcut to launch the app

---

#### **Method B: Manual Shortcut Creation**

1. Right-click on your **Desktop**
2. Select **"New" → "Shortcut"**
3. Click **"Browse..."**
4. Navigate to your extracted folder
5. Select **`Karachi Garment Store POS.exe`**
6. Click **"Next"**
7. Name the shortcut: `Karachi Garments Store POS`
8. Click **"Finish"**

✅ **Done!** Shortcut is now on your desktop

---

#### **Method C: Using Command Prompt (Advanced)**

Open Command Prompt as Administrator and run:

```powershell
cd "%USERPROFILE%\Desktop"
powershell -Command "$WshShell = New-Object -ComObject WScript.Shell; $shortcut = $WshShell.CreateShortcut('Karachi Garments Store POS.lnk'); $shortcut.TargetPath = 'C:\Path\To\Karachi-Garments-Store-POS-v1.0.0\Karachi Garment Store POS.exe'; $shortcut.WorkingDirectory = 'C:\Path\To\Karachi-Garments-Store-POS-v1.0.0'; $shortcut.IconLocation = 'C:\Path\To\Karachi-Garments-Store-POS-v1.0.0\resources\app\public\logo.png'; $shortcut.Save()"
```

Replace `C:\Path\To\` with your actual extracted folder path.

---

### **Step 5: Customize Shortcut (Optional)**

#### **Change Shortcut Icon:**

1. Right-click the shortcut on desktop
2. Select **"Properties"**
3. Click **"Change Icon..."**
4. Browse to the extracted folder → `resources/app/public/logo.png`
5. Click **"OK"** → **"Apply"** → **"OK"**

#### **Run as Administrator (Recommended for barcode server):**

1. Right-click the shortcut
2. Select **"Properties"**
3. Click **"Advanced..."**
4. Check **"Run as an administrator"**
5. Click **"OK"** → **"Apply"** → **"OK"**

---

### **Step 6: Create Start Menu Entry (Optional)**

1. Extract folder location: `C:\Users\YourUsername\Karachi-Garments-Store-POS-v1.0.0\`
2. Open File Explorer
3. Navigate to: `C:\ProgramData\Microsoft\Windows\Start Menu\Programs\`
4. Create new shortcut (right-click → New → Shortcut)
5. Point to: `C:\Users\YourUsername\Karachi-Garments-Store-POS-v1.0.0\Karachi Garment Store POS.exe`
6. Name it: `Karachi Garments Store POS`
7. Save

Now the app will appear in Windows Start Menu!

---

### **Step 7: Pin to Taskbar (Optional)**

1. Right-click the desktop shortcut
2. Select **"Pin to taskbar"**

Now the app is one-click away from your taskbar!

## 🔧 **Folder Placement Recommendations**

### **Option 1: User AppData (Recommended)**
```
C:\Users\YourUsername\AppData\Local\Karachi-Garments-Store-POS-v1.0.0\
```
✅ No admin rights needed to move/modify  
✅ App data saved per user  
✅ Easy to backup

### **Option 2: Program Files**
```
C:\Program Files\Karachi-Garments-Store-POS-v1.0.0\
```
✅ Standard installation location  
⚠️ Requires admin rights  
⚠️ Shared across all users

### **Option 3: Root Folder (Simple)**
```
C:\Karachi-Garments-Store-POS-v1.0.0\
```
✅ Easy to access  
⚠️ Messy for multiple apps

## 📱 **Barcode Scanner Setup (Post-Deployment)**

Once app is running on the system:

1. Open **Settings → Barcode Scanner Setup** in the POS app
2. Note the **Server IP** and **Port** (3001)
3. On Android/iPhone barcode scanner app:
   - Set webhook URL to the IP shown
   - Test by scanning a barcode
4. Barcode should appear in the POS app

## 🛠️ **Troubleshooting**

| Issue | Solution |
|-------|----------|
| **EXE won't start** | Try running as administrator |
| **Splash screen stuck** | Wait 10 seconds, restart app |
| **Database error** | Delete `app-data.db`, restart app |
| **Barcode not working** | Check same WiFi, verify server URL |
| **App crashes** | Check Windows Event Viewer, reinstall |
